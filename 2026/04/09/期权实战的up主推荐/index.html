期权实战的up主推荐

一、国内实战派（A 股 / 商品期权，首选）

1. 发鹏期权说（B 站 / 公众号）

  • 风格：专业机构派，国内期权元老级人物。
  • 核心内容：中性卖方、波动率曲面、铁鹰 / 蝶式优化、反脆弱风控。内容极其硬核，擅长期权量化分析，是学习高胜率卖方策略（如铁鹰）的顶级资源。
  • 适合：进阶学习者，想建立科学、系统的期权交易体系。

2. 期权 Daily 日记（B 站）

  • 风格：散户实战派，真实复盘，不吹不黑。
  • 核心内容：分享小额实盘账户交易记录，重点讲解铁鹰、日历价差等中性策略。视频《期权新手必看：5 个让我亏 80% 的蠢错误》堪称经典，风险控制理念极强。
  • 适合：新手及实战学习者，接地气、避坑首选。

3. 交易艺术汇（陈竑廷，B 站 / 公众号）

  • 风格：学院派 + 实战派，清华背景。
  • 核心内容：主讲期权增强收益、降低尾部风险，系统讲解各类价差策略（含铁鹰），擅长用期权做资产保护，内容严谨且有深度。
  • 适合：追求稳健、低回撤的投资者。

4. 黄旭东（B 站 / 课堂）

  • 风格：大赛冠军派，全国期货实盘大赛期权组亚军。
  • 核心内容：龙头追踪法、实战心态管理、震荡市策略。讲课极度接地气，干货密集，适合想快速上手实盘的交易者。
  • 适合：喜欢短线 + 波段结合期权的交易者。

二、美股 / 全球市场派（适合学习标准策略）

1. 美投讲美股（美投君，B 站）

  • 风格：学术派，逻辑清晰。
  • 核心内容：系统期权学堂（从入门到希腊字母），标准铁鹰、裸卖空、对冲保险策略。用大量图表拆解收益结构，理论扎实，适合打基础。
  • 适合：建立完整知识框架，学习标准期权策略。

2. 小马哥 0701（B 站）

  • 风格：CFA 持证人，专业严谨。
  • 核心内容：金融衍生品公开课，期权定价、希腊字母、高级策略深度解析。适合想吃透底层原理的学习者。

三、学习建议（按你的铁鹰策略）

  1. 入门筑基：先看 美投君 或 小马哥 0701，搞懂铁鹰的标准结构、风险收益、希腊字母。
  2. 实战避坑：再看 期权 Daily 日记，学习实盘操作、仓位管理、心态控制，少走弯路。
  3. 进阶优化：最后深入 发鹏期权说，学习波动率择时、铁鹰策略变体、风控模型，提升胜率与盈亏比。

⚠️ 风险提示：期权交易风险极高，策略回测不代表未来收益。以上推荐仅为学习参考，不构成任何投资建议。请务必先用模拟盘熟练策略后，再考虑实盘。

铁鹰策略实战・UP 主必看清单

只给最硬核、最贴近铁鹰实战的视频，不浪费时间。

1）发鹏期权说（进阶天花板，必看）

  • 《波动率曲面与铁鹰开仓时机》

    教你：什么时候开铁鹰胜率最高、IV 怎么看

  • 《铁鹰策略风控：移动止损 vs 宽腿》

    实战风控，避免大亏

  • 《蝶式 vs 铁鹰：怎么选胜率更高》

    刚好对应你现在学的内容

2）期权 Daily 日记（实盘避坑，必看）

  • 《铁鹰策略完整实盘：开仓→调仓→平仓》

    一步一步跟着做

  • 《期权 5 个让我亏 80% 的错误》

    铁鹰最容易死在这几条

  • 《卖方仓位管理：永不爆仓公式》

    铁鹰核心就是仓位

3）美投讲美股（标准铁鹰基础，必看）

  • 《Iron Condor 铁鹰策略：原理 + 收益图》

  • 《铁鹰开仓参数：行权价间距、到期日》

  • 《波动率高 / 低怎么选策略》

    最标准教科书，打地基用

4）交易艺术汇（稳健增强，必看）

  • 《中性策略：铁鹰如何降低回撤》
  • 《期权卖方的风险：你以为的有限≠真的有限》
  • 《宽跨 vs 铁鹰：为什么 90% 人选错》
