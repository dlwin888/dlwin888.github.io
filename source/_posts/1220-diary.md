### 1220日记

1. SpringBoot-Learning (程序猿DD)

   **地址：** GitHub 上搜索 `dyc87112/SpringBoot-Learning`

   **特点：** 这是一个非常经典的开源项目，包含了Spring Boot 2.x的基础教程，质量很高，可以作为工程脚手架参考

2. springBoot (集合类项目)

   - **地址：** GitHub 上搜索 `527515025/springBoot`
   - **特点：** 包含了大量组件的整合示例（如JPA, MyBatis, WebSocket, Security, Shiro, Cache等），拿来即用，非常适合遇到具体问题时查阅代码。

3. springboot-project (点餐系统)

   - **地址：** GitHub 上搜索 `sqmax/springboot-project`
   - **特点：** 一个基于微信服务号的点餐系统后端，前后端分离，非常适合想做全栈开发或毕业设计的同学参考。
