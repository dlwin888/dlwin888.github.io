### 260425日记

1. #### [animate.css](https://github.com/animate-css/animate.css) 

   [一个开源 CSS 动画库](https://animate.style/)。提供大量现成的网页动画效果，例如淡入淡出、弹跳、滑动、旋转等。只需要给元素添加对应的 CSS 类名，就可以快速实现动画效果，不需要编写复杂的动画代码。

2. #### [vue-naive-admin](https://github.com/zclzone/vue-naive-admin)

   [项目说明](https://www.isme.top/)：一款极简风格的基于 Vue3 + Vite + Pinia + Unocss + Naive UI 的轻量级后台管理模板。

3. #### [css-generators](https://css-generators.com/)

   一个在线 CSS 代码生成器集合网站，快速创建各种 CSS 样式代码。它提供了很多实用的小工具（如形状、分隔线、渐变、加载动画、边框、工具提示等 CSS 效果生成器。

4. #### [WeChat Markdown Editor](https://github.com/doocs/md)

   一款高度简洁的微信 Markdown 编辑器，自动即时渲染为微信图文，支持 Markdown 语法、自定义主题样式、内容管理、多图床、AI 助手等特性。
