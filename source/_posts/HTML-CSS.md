### **3. HTML + CSS**

#### **HTML**

HTML 构建了网络上的内容，例如文本、链接和表单。可以将其视为网页的骨架。

**示例**：使用`<form>`和`<input>`创建登录表单。

![HTML](../uploads/v2-d2af95176ab3476912135553e94151b3_r.jpg)

#### **CSS**

CSS 设置 HTML 样式，控制布局、颜色和间距。

**示例**：使用 Flexbox 将元素居中或使用 Grid 创建多列布局。

![CSS](../uploads/v2-7943a1d90c02ac6e3664c663760de141_1440w.jpg)
