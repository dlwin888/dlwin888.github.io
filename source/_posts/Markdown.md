Markdown（md）是一种轻量级的标记语言，用于格式化文本下面是一些常用的Markdown法：

### 1. 标题使用 `#` 符号来创建标题。`#` 的数量表示标题的层级（从1到6）。



```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 2. 段落

段落由一个或多个行组成，段落之间需要有一个空行。



```markdown
这是一个段落。

这是另一个段落。
```

### 3. 强调

使用 `*` 或 `_` 来添加强调。



```markdown
*斜体文本* 或 _斜体文本_

**粗体文本** 或 __粗体文本__

***粗斜体文本*** 或 ___粗斜体文本___
```

### 4. 列表

无序列表使用 `*`、`-` 或 `+`。



```markdown
- 项目一
- 项目二
  - 子项目一
  - 子项目二
```

有序列表使用数字加点。



```markdown
1. 第一项
2. 第二项
   1. 子项一
   2. 子项二
```

### 5. 链接

使用 `[]` 和 `()` 来创建链接。



```markdown
[链接文本](http://example.com)
```

### 6. 图片

图片的语法与链接类似，在前面加一个 `!`。



```markdown
![替代文本](http://example.com/image.jpg)
```

### 7. 引用

使用 `>` 来添加引用。



```markdown
> 这是一个引用。
```

### 8. 代码

使用 ``` 符号来标记行内代码。



```markdown
这是 `行内代码` 示例。
```

使用三个 ``` 来创建代码块。



```markdown

```

```

```

### 9. 分隔线

使用三个以上的 `*`、`-` 或 `_` 来创建分隔线。



```markdown
---
```

### 10. 表格

使用 `|` 来创建表格。



```markdown
| 列1 | 列2 |
|-----|-----|
| 数据1 | 数据2 |
| 数据3 | 数据4 |
```

这是一些基本的Markdown语法，使用这些语法可以轻松地格式化文本。希望对你有帮助！
