## 整体学习路径说明

`@knadh/oat` 是轻量级无依赖 UI 库，核心特点是：类名简洁（如 `oat-btn`/`oat-input`）、仅提供基础样式、需结合原生 HTML/CSS 实现复杂布局，非常适合新手聚焦「布局 + 样式」核心能力。

|  梯度  |         核心目标         |              掌握的关键技能              |
| :----: | :----------------------: | :--------------------------------------: |
| 入门级 | 熟悉 Oat UI 基础组件使用 |   组件引入、基础布局（Flex）、简单交互   |
| 进阶级 |  整合多组件实现完整功能  |  组件组合、本地存储、条件渲染、事件处理  |
| 综合级 |     模拟真实开发场景     | 复杂布局、响应式适配、生态整合（如图表） |
