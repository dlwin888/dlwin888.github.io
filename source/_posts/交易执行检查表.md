### 螺纹钢 RB 日内卖方期权・交易执行检查表

适用：**熊市看涨价差 (Bear Call) / 牛市看跌价差 (Bull Put)**

风格：日内、Theta 收益、稳曲线、不隔夜

#### 一、开仓前必查（全部打√才能开）

- [ ] 日线处于**震荡区间**，无明显单边趋势
- [ ] 日内波动温和，无放量突破 / 无极端消息
- [ ] 期权到期剩余：**7～14 天**
- [ ] 只做一个方向：不同时做 Call+Put 价差
- [ ] 总资金仓位 ≤ **5%**
- [ ] 今日未触发单日亏损上限

#### 二、开仓点位标准（现价≈3700）

#### 偏空震荡：Bear Call Spread

- [ ] 卖出：**RB-C-3700**
- [ ] 买入：**RB-C-3750**
- [ ] 行权间距：**50 点**

#### 偏多震荡：Bull Put Spread

- [ ] 卖出：**RB-P-3700**
- [ ] 买入：**RB-P-3650**
- [ ] 行权间距：**50 点**

#### 三、持仓中监控

- [ ] 价格未突破强弱边界（3650 / 3750）
- [ ] 波动率未异常飙升
- [ ] 未出现连续反向大 K 线
- [ ] 仓位始终 ≤ 2 手

#### 四、止盈条件（满足任意一个立即平仓）

- [ ] 盈利达到**净权利金的 60%**
- [ ] 持仓已**1 个交易日**
- [ ] 收盘前 **15 分钟**（强制平仓）
- [ ] 价格到达最优盈利区间

#### 五、止损条件（满足任意一个立即平仓）

- [ ] 价格**有效突破 3650 或 3750**
- [ ] 浮亏达到**最大风险的 40%**
- [ ] 开盘 1 小时仍未盈利
- [ ] 单日亏损达总资金 **1%**

#### 六、加仓唯一条件（缺一不可）

- [ ] 已有持仓浮盈 ≥ **50% 目标收益**
- [ ] 行情仍在震荡区间内
- [ ] 加仓后总手数 ≤ **2 手**

#### 七、收盘铁律

- [ ] 今日所有持仓**全部平仓**
- [ ] 不留任何仓位隔夜
- [ ] 记录当日盈亏与失误
